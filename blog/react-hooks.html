# 使用 React Hooks 节省 90% 的代码

react hooks 是 React 16.8 的新增特性。 它可以让我们在函数组件中使用 state 、生命周期以及其他 react 特性，而不仅限于 class 组件。

react hooks 的出现，标示着 react 中不会在存在无状态组件了，只有类组件和函数组件。
